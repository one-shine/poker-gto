# プライバシーポリシー (ドラフト)

> **状態: ドラフト。** 公開前に「事業者名」「連絡先」「施行日」を確定し、配信形態
> (Web / PWA / アプリストア) と収益化方針 (広告・解析の有無) に応じて該当節を更新すること。
> 解析・広告・アカウント機能を追加する場合は、本ポリシーと同意管理 (CMP / ATT) を必ず見直す。

本アプリ (以下「本アプリ」) は、ポーカーの GTO (ゲーム理論的最適) 戦略を学習するための
シミュレーション/学習ツールです。本ポリシーは本アプリが取り扱う情報を説明します。

## 1. 基本方針 — データは端末内に留まります

本アプリは **アカウント登録不要** で、利用者の学習データは原則 **利用者自身の端末内にのみ**
保存されます。本アプリの運営者がそれらを収集・送信・閲覧することはありません。

## 2. 端末内に保存される情報

本アプリは以下を端末のブラウザストレージ (localStorage / IndexedDB) に保存します。
いずれも本人の学習体験のためのもので、外部送信されません。

| 保存キー | 保存先 | 内容 |
|---------|--------|------|
| `poker-gto-settings` | localStorage | ゲーム設定 (スタック・モード・自動送り秒数・チュートリアル完了フラグ等) |
| `poker-gto-progress` | localStorage | 学習進捗 (XP・レベル・UI 表示レベル・ミス履歴) |
| `poker-gto-session` | IndexedDB | セッション統計 (精度・ミス記録・ハンド履歴) |

- これらは **個人を特定する情報を含みません** (氏名・メール・位置情報・連絡先などは収集しません)。
- 端末・ブラウザのサイトデータを削除すると、これらの学習データは完全に消去されます。

## 3. 外部との通信

本アプリは既定では学習データを外部送信しませんが、以下の限定的な外部通信があります。

1. **Web フォントの読込 (Google Fonts)**
   UI フォントを Google Fonts の CDN から読み込みます。この際、フォント配信元 (Google) の
   サーバーに利用者の IP アドレスやブラウザ情報が技術的に到達します (Google のプライバシー
   ポリシーが適用されます)。これを避けたい場合はフォントのセルフホスト構成を選べます。

2. **クラッシュレポート (任意・既定で無効)**
   不具合解析のためのクラッシュレポート送信機構を備えますが、**配布者が送信先 (Sentry DSN)
   を明示的に設定した場合にのみ有効**になります。既定では送信は行われず、エラーはブラウザの
   コンソールに記録されるのみです。有効化する場合は、送信内容と送信先を本ポリシーに追記します。

現時点で本アプリは **広告・行動トラッキング・第三者解析 (Google Analytics 等) を使用しません。**
将来導入する場合は本ポリシーを更新し、必要な同意取得 (EU: 同意バナー / iOS: ATT) を実施します。

## 4. Cookie について

本アプリはトラッキング目的の Cookie を使用しません。データ保存には localStorage / IndexedDB
のみを用い、いずれも本人の学習体験の維持に限定されます。

## 5. 第三者提供

本アプリの運営者は、端末内の学習データを第三者へ提供・販売しません (そもそも取得しません)。

## 6. 子どもの利用について

本アプリは賞金や実マネーのやり取りを伴わない **学習/シミュレーション** ですが、ポーカーを
題材とするため対象年齢を **17歳以上 (Teen 以上)** とします。本アプリは「ギャンブル」ではなく、
実際の賭博・換金機能を一切提供しません。

## 7. データの管理・削除

利用者は、ブラウザまたは OS のサイトデータ/アプリデータ削除機能を用いて、いつでも本アプリの
保存データを全消去できます。本アプリの運営者側に保存されるデータはありません。

## 8. 準拠法・将来の対応

将来、外部送信を伴う機能 (アカウント・クラウド同期・解析・広告) を追加する場合は、対象地域の
法令 (例: EU GDPR、米国 CCPA/CPRA) に従い、同意取得・開示・削除請求対応を整備します。
