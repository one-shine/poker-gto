## コンセプト

カジノの派手さではなく **プレミアムなポーカー分析端末**。落ち着いた深いダーク
+ フェルトグリーン + 真鍮(ブラス)アクセント。データ(BB/%/EV)は等幅フォントで
「端末」感を出す。記憶に残る一点は **照明付きフェルトテーブル + ブラスの質感**。

- トーン: 上質 / 分析的 / 集中。減点法ではなく意図のある洗練。
- 1番の差別化: 照明付きフェルト + ブラス + 等幅データ表示。

## デザイントークン

### フォント(Google Fonts)
| 用途 | フォント |
|------|---------|
| 見出し/ディスプレイ | Bricolage Grotesque |
| 本文(欧文) | Hanken Grotesk |
| 本文(和文) | Zen Kaku Gothic New |
| 数値データ | JetBrains Mono |

### カラーパレット
- 背景: 緑みを帯びた深いチャコール(階層を付ける。純黒は使わない)
- フェルト: 中心が明るいラジアル(`#1f5c43` 付近 → 縁 `#0c3325`)+ インナーシャドウ + グレイン
- アクセント: ブラス/ゴールド(`#d4af37` 系)— ディーラーボタン / 手番リング / プレミアム表現
- アクション: レイズ=エメラルド、フォールド=クレイレッド(必ずアイコン併用)、コール=クール系
- カード面: アイボリー(`#f5f1e8` 系・純白でない紙質)/ 赤スート=深いバーミリオン / 黒スート=インク

### カード描画の最終方針
本物のトランプのフル・ピップは視認性を落とすため**廃止**。他のポーカーアプリ同様に
**「大きなランク + スート記号」のクリーン表示**(中央に rank=display フォント極太 + 直下に suit 記号)。
色 + スート記号併用で色覚配慮。裏面はブラス菱格子 + 「G」エンブレム。

## 厳守する制約

- **テスト契約を壊さない**: `role="img"` / `aria-label`(「A スペード」「コール 1BB (c)」「3BB (3BB)」「ベット額スライダー」等)/ 表示テキスト(「手番中」「レイズ 2.5BB」「オールイン」「ポット {n}BB」)を維持。テストは構造ではなくこれらの契約に依存。ポットラベルは単一テキストノードのまま(分割すると `getByText` が壊れる)。
- **色のみ非依存**([`./SPEC.md`](./SPEC.md) アクセシビリティ): スート記号・形状・アイコンを色と必ず併用。
- **モバイル対応**: rem/%/vw、タップターゲット 44px 維持。`useIsMobile`/`useContainSize` で卓を流動レスポンシブ化。
- `src/engine/` は触らない(React 非依存維持)。
- Framer Motion は描画後の aria/テキストを変えない範囲で使用。
